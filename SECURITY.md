# Security Policy

## Supported Versions

| Branch  | Supported          |
| ------- | ------------------ |
| main    | :white_check_mark: |
| dev     | :white_check_mark: |

## Reporting a Vulnerability

We ask that all suspected vulnerabilities be privately and responsibly disclosed by [contacting our security team](mailto:security+amazeeai@amazee.io).

## Communication

[GitHub Security Advisories](https://github.com/amazeeio/amazee.ai/security/advisories) will be used to communicate during the process of identifying and resolving the vulnerability.

The advisory will only be made public when the patched version is released to inform the community of the vulnerability and its potential security impact.
